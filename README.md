# movies-explorer-api

## Серверная часть дипломного проекта.

## Функциональность:

---

- Регистрация,
- Авторизация,
- Обновление данных пользователя,
- Получение информации о текущем пользователе,
- Получение списка фильмов,
- Создание фильма,
- Удаление фильма,
- Обработка ошибок,
- Валидация входящих данных.

## Стек технологий:

---

- MongoDB,
- Express,
- nodemon,
- mongoose,
- dotenv,
- cors,
- celebrate,
- bcryptjs,
- winston,
- express-winston,
- helmet,
- jsonwebtoken,
- validator,
- eslint.
